# 🛡️ Datatrux - AI Scam Detection System

## 📌 Overview

Datatrux is an AI-powered scam detection web application designed to identify phishing URLs, scam websites, and fraudulent emails using Machine Learning techniques. The application provides users with a simple interface to analyze suspicious links and messages, helping improve online safety.

---

## ✨ Features

- 🔍 Detect phishing URLs
- 📧 Analyze suspicious emails
- 🌐 Identify fraudulent websites
- 🤖 Machine Learning-based predictions
- 💻 User-friendly Flask web interface
- 📊 Fast and accurate scam detection

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/datatrux-ai-scam-detection.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

---

## 🚀 Usage

1. Launch the Flask application.
2. Enter a suspicious URL or email.
3. Click **Detect**.
4. View the prediction result.
